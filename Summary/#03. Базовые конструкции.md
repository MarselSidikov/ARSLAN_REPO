# Алгоритм

* Последовательность действий, которые отвечают требованиям:

1) Ввод
2) Вывод
3) Эффективность
4) Однозначность
5) Конечность

* Алгоритм может быть выражен тремя конструкциями:

1) Линейное исполнение:

```
  [A]
   |
   V
  [B]
   |
   V
  [C]
   |
   V
```

2) Условие

```
 (+)-<E>-(-)
  |       |
  V       V
 [A]     [B]
  ---------
      |
      V
````

```JAVA
if (E) {
	A;
} else {
	B;
}
```

3) Цикл

```
-(-)-<E><--|
|    (+)   |
|     |    |
|     V    |
|    [A]----
|
V
```

```JAVA
while (E) {
	A;
}
```
